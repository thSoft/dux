## Contributor Guide

### Setup

* Install [sbt](http://www.scala-sbt.org/release/docs/Setup.html)
* Clone this repository
* Run `scripts/setup`

### Setup IDE

* Install [Eclipse Scala IDE](http://scala-ide.org/download/sdk.html)
* Import existing projects into workspace

### Develop

* Run `scripts/develop`
* Open `index.html`
